Páginas web

Características de una página web

  1. Contenido estructurado
  2. Acceso a través de internet
  3. Conexión con otras páginas
  4. Interactividad
  5. Ubicación en un sitio web

Tipos de páginas web

  • Estáticas

    • Su contenido no cambia a menos que el desarrollador lo modifique manualmente

  • Dinámicas

    • Su contenido se genera automáticamente o puede cambiar según la interacción del usuario

¿Cómo funciona una página web?

  1. Un usuario ingresa a la URL en su navegador
  2. El navegador solicita la página web al servidor donde está almacenada
  3. El servidor responde enviando los archivos (HTML, CSS, JavaScript, etc.)
  4. El navegador procesa estos archivos y muestra la página al usuario
