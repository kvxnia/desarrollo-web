CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir el diseño y la presentación de documentos web escritos en HTML o XML. Mientras que HTML define la estructura y el contenido de una página, CSS se encarga de su apariencia, permitiéndote controlar aspectos como colores, fuentes, márgenes, bordes.

¿Qué es un selector?

Un selector en CSS es un componente que identifica los elementos HTML a los que deseas aplicar estilos. Es la primera parte de una regla de CSS y actúa como una "dirección" para que el navegador sepa qué elementos modificar.
