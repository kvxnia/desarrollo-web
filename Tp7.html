CLASES

Definición

Las clases en CSS son una forma de agrupar elementos HTML y aplicarles estilos específicos. Imagina que tienes varios párrafos en una página web y quieres que todos tengan el mismo color de fuente, tamaño y espaciado. En lugar de repetir los mismos estilos para cada párrafo, puedes crear una clase y asignarla a todos ellos

¿Cómo funcionan las clases?

  1. Creación de la clase

    2. En tu archivo CSS, defines una clase asignándole un nombre único, precedido de un punto (.)

  2. Aplicación de la clase

    3. En tu archivo HTML, asignas la clase a los elementos que quieres que tengan esos estilos. Se hace utilizando el atributo class dentro de la etiqueta del elemento

Ventajas de usar clases

  • Reutilización de estilos
  • Facilidad de mantenimiento
  • Organización del código
  • Flexibilidad
